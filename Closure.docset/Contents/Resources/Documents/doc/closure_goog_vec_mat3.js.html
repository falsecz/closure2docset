mat3.js
Implements 3x3 matrices and their related functions which are compatible with WebGL. The API is structured to avoid unnecessary memory allocations. The last parameter will typically be the output vector and an object can be both an input and output parameter to all methods except where noted. Matrix operations follow the mathematical form when multiplying vectors as follows: resultVec = matrix * vec.
