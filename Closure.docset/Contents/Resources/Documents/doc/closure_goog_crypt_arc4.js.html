arc4.js
ARC4 streamcipher implementation. A description of the algorithm can be found at: http://www.mozilla.org/projects/security/pki/nss/draft-kaukonen-cipher-arcfour-03.txt. Usage: var arc4 = new goog.crypt.Arc4(); arc4.setKey(key); arc4.discard(1536); arc4.crypt(bytes); Note: For converting between strings and byte arrays, goog.crypt.base64 may be useful.
