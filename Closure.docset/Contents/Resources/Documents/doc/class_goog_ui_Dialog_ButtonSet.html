ui.Dialog.ButtonSet Extends goog.structs.Map
A button set defines the behaviour of a set of buttons that the dialog can show. Uses the goog.structs.Map interface.

numericKeyMap_ :

A map of internal keys that are numeric and should be cast back to a number on retrieval. A previous implementation had optimizations for IE6's bad GC. However, the keys weren't correctly cast to strings so people started to depend on the behavior of numeric keys in tests. To ensure backwards compatibility this part remains, but it would be nice to strip out entirely. See http://b/5622311.
