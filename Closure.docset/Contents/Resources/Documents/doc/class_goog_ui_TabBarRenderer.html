ui.TabBarRenderer Extends goog.ui.ContainerRenderer
Default renderer for goog.ui.TabBars, based on the TabPane code. The tab bar&#39;s DOM structure is determined by its orientation and location relative to tab contents. For example, a horizontal tab bar located above tab contents looks like this:
