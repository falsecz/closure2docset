a11y.js
Utilities for adding, removing and setting ARIA roles as defined by W3C ARIA Working Draft: http://www.w3.org/TR/2010/WD-wai-aria-20100916/ All modern browsers have some form of ARIA support, so no browser checks are performed when adding ARIA to components.
