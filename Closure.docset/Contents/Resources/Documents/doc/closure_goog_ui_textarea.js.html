textarea.js
A content-aware textarea control that grows and shrinks automatically. This implementation extends goog.ui.Control. This code is inspired by Dojo Dijit&#39;s Textarea implementation with modifications to support native (when available) textarea resizing and minHeight and maxHeight enforcement.
