goog.silverlight.ClipboardButton
A button that can access the native clipboard, via Silverlight. If this is not a browser that supports native clipboard access, throw an error. Clients should not instantiate this directly. Instead, use CopyButton or PasteButton.
goog.silverlight.ClipboardEvent
A clipboard button event. By design, replicates IE's clipboard API.
goog.silverlight.CopyButton
A button that can copy contents.
goog.silverlight.PasteButton
A button that can retrieve contents from the native clipboard.
