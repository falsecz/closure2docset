hsvpalette.js
An HSV (hue/saturation/value) color palette/picker implementation. Inspired by examples like http://johndyer.name/lab/colorpicker/ and the author's initial work. This control allows for more control in picking colors than a simple swatch-based palette. Without the styles from the demo css file, only a hex color label and input field show up.
